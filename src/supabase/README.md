## 🚀 Quick Setup (3 Steps)

1. **Run SQL Migration** (5 min)
   - Open `/SUPABASE_COMPLETE_SETUP.md`
   - Copy SQL from "Complete SQL Migration" section
   - Paste in Supabase SQL Editor
   - Click "Run"

2. **Create Storage Buckets** (2 min)
   - Follow instructions in setup guide
   - Create 5 buckets in Supabase Dashboard

3. **Initialize Database** (1 min)
   - Run initialization script
   - Creates admin account + sample data

**Total setup time: ~8 minutes**
